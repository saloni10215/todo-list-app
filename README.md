# To-Do List App

## About

This is a modern To-Do List web application built using HTML, CSS, and JavaScript. It helps users manage daily tasks efficiently with task tracking, editing, completion status, and local storage support.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Undo completed tasks
- Task completion persists after page refresh
- Save tasks using Local Storage
- Task Counter
- Empty State Message when no tasks exist
- Enter Key support for adding tasks quickly
- Responsive and clean user interface
- Dark Mode with theme persistence

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## How to Use

1. Enter a task in the input field.
2. Click **Add** or press **Enter**.
3. Click a task to mark it as completed.
4. Click **Edit** to modify a task.
5. Click **Delete** to remove a task.
6. Tasks and completion status are automatically saved in the browser.
7. Use **Dark Mode** to switch between light and dark themes.
